## 技术栈

- 纯 HTML + CSS + JavaScript
- 响应式设计
- 无需后端

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/863king/ai2091-nav.git
cd ai2091-nav

# 直接打开 index.html 或使用本地服务器
python -m http.server 8080
```
